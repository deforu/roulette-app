# roulette-app
ルーレットWebアプリ

[ルーレットアプリのURL](https://roulette-app-git-main-deforus-projects.vercel.app/)

使い方
1. index.html をブラウザで開きます。
2. 必要に応じて「現在のジャンル」の鉛筆ボタンからジャンル名と候補を編集し、`✓`で保存します。新規登録は「＋ ジャンルを追加」を押します。
3. 「ルーレットを回す」を押します。
4. 結果を確定する場合は「決定」、候補に残す場合は「保留」を押します。

入力内容、ジャンル設定、選択済み履歴、サウンド設定はブラウザに自動保存され、次回のページ読み込み時に復元されます。

動作
- 決定: 選ばれた項目を候補から削除し、画面下部の「選択済み」へ移動します。
- 保留: 選ばれた項目を候補に残します。
- 初期化: 候補と選択済み一覧を初期状態へ戻します。
